Tech stack

Built with

SparseViewDiT

Diffusion transformer that generates 16 novel views from 1–4 sparse input images using Flow Matching

NVIDIA Research

Object TokenGS

Feedforward encoder-decoder that lifts multiview images into 3D Gaussians in a single forward pass

NVIDIA Research

YOLOv8n

Real-time object detection for extracting vehicle and pedestrian crops from dashcam frames

Ultralytics

3D Gaussian Splatting

Represents 3D scenes as millions of Gaussian ellipsoids — photorealistic, real-time renderable

Kerbl et al. 2023

ffmpeg

Frame extraction from dashcam video at configurable FPS with hardware acceleration support
